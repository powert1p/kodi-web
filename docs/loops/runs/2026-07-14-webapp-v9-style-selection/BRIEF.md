# AiPlus web — раунд выбора визуального направления

## Что выбираем

Пять rendered-направлений для одного и того же учебного сценария. Это не пять палитр на одном
wireframe: у каждого варианта своя пространственная модель. После выбора один вариант переносится
в React PWA и становится основой нового канона.

## Frozen input

- Продукт: индивидуальная AI-практика по математике для школьников 10–13 лет.
- Бренд: настоящий знак AiPlus из `assets/aiplus-logo-reference.jpg`.
- Маскот: реальный бежевый персонаж в синем свитшоте из
  `assets/aiplus-mascot-reference.jpg`; не придумывать нового героя и не использовать старый SVG.
- Фирменный акцент: orange/black/white. Оранжевый — действие и подпись бренда, а не основной фон UI.
- Old design system — anti-reference и источник полезных качеств: сохранить аккуратность,
  типографическую иерархию и тёплую бумагу; не возвращать официальную строгость, полноэкранную
  клетку, декоративный маршрут или чёрный hero-блок.
- Duolingo — ориентир по ясности следующего действия, спокойной геймификации и роли маскота,
  не визуальный шаблон.

## Одинаковый сценарий для сравнения

- Тема: «Проценты».
- Прогресс: шаг 1 из 4.
- Задача: «Цена 1200 ₸ выросла на 15%, затем снизилась на 10%. Какой стала цена?»
- Текущий шаг: `1200 × 0,15 = ?`.
- Реплика маскота: «Сначала найдём прибавку — не итоговую цену».
- Одно основное действие: «Проверить».

## Hard gates

1. Математика занимает не менее 60% первого viewport и остаётся первым объектом внимания.
2. Насыщенный orange занимает не более 8% viewport; нет neon/acid/многоцветного dashboard.
3. Один реальный маскот, расположенный вне math-canvas; он помогает, но не закрывает условие.
4. Один primary CTA; touch-target не меньше 44×44 px; видимый keyboard focus.
5. Нет горизонтального overflow на 375 px и 1280 px.
6. Учебный текст не меньше 18 px; metadata не меньше 13 px и с читаемым контрастом.
7. Концепты различаются композицией: studio / path / board / club / poster.
8. Никаких изменений production React до выбора владельца продукта.

## Варианты

1. **AiPlus Studio** — центральная рабочая сцена и компактное coach-окно.
2. **Живой маршрут** — короткий data-backed lesson path раскрывается в текущую задачу.
3. **Белая доска** — единая math-board без карточного зоопарка.
4. **Клуб AiPlus** — мягкое пространство с отдельной нишей тренера и спокойной math-stage.
5. **Живой плакат** — асимметричная editorial-композиция, где формула является hero.
