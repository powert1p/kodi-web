# Benchmark contract — webapp v10

- Objective: premium mobile-first redesign of Kodi/AiPlus while preserving product behavior.
- Frozen inputs: `RESEARCH.md`, `BRIEF.md`, `RUBRIC.md`, real logo/mascot assets and product state contract.
- Anti-reference: current project DS, v7/v8 implementation, all five v9 concepts and generated seed/fox SVG.
- Unit of comparison: same Hub/Drill/Feedback content, same device sizes, same reduced-motion setting.
- Divergence gate: at least three meaningfully different spatial systems before any production DS rewrite.
- Selection: independent screenshot judges; no direction may win from self-description alone.
- Terminal condition: `RUBRIC.md` READY threshold + green mechanical gate.
- Stop policy: user explicitly requested a loop until quality bar; no arbitrary round count. A round ends only with immutable renders, scores and deltas.
- Change discipline: fix consensus deltas; discard a weak macro-direction instead of accumulating decorative patches.
