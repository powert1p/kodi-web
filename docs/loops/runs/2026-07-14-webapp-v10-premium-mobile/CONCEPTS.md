# Concept round — clean slate

Три направления собраны из одного frozen brief и одних production данных. Старые
`DESIGN_SYSTEM.md`, V7, V8 и V9 использовались только как anti-reference.

## 1. Atelier / solution thread

Editorial rhythm, Literata + Onest, открытая очередь и оранжевая нить как ход
решения. Сильнее всего по craft и спокойствию, но headline иногда забирает фокус у
математики.

## 2. Equality / living math object

Равенство — главная рабочая поверхность. Активная неизвестная живёт внутри
оранжевой скобки, а ответ после проверки встраивается в proof chain. Сильнее всего
по product distinctiveness и math focus.

## 3. CUT / cinematic seam

Две контрастные плоскости и диагональный image seam. Самое campaign-like
направление: выразительное, но слишком рекламное и хрупкое для длинной математики.

## Решение

Production kernel: **Equality interaction model + Atelier material and type
restraint**. Из CUT допускается только редкий image seam в transition/closure.

Причина: два из трёх reviewers выбрали Equality, а третий выбрал Atelier; все трое
независимо предложили этот же гибрид и назвали одинаковые blockers: long math,
полная state grammar и смысловая motion signature.
