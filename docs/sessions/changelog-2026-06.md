## 2026-06-23 16:00
**Тип:** plan (новая фича + движок)
**Зачем:** «работа над ошибками» для НИШ-срезов — кабинет, где ребёнок разбирает заваленную задачу по подзадачам, ведущим к ПОНИМАНИЮ (не выдавая ответ); при затыке — задача попроще; рядом AI-тьютор. Старый Flutter-флоу владельцу не нравится.
**Что сделано:**
- **Новый фронт-кабинет** `cabinet/` (React 19 + TS + Vite SPA + Tailwind v4 + KaTeX + Zustand, HashRouter) — отдельно от Flutter, embeddable в мобилку через WebView позже. 5 экранов: Срез-хаб → Лесенка разбора (signature: ступени-«рунги», gating) → AI-тьютор (bottom-sheet, mock) → Закрытие. Mock-данные по форме backend-декомпозиции (тема→микро-навык→задача-в-шагах).
- **Reasoning-first лесенки:** ступень = ОДНА мысль; типы `compute` (ввод) и `choose` (выбор-развилка под ловушку); `reveal` (арифметика) — только при затыке; climb-down «задача попроще» при 2 ошибках.
- **Реальные НИШ-задачи** из базы (`source != generated`): проценты/множества/отношения/работа/движение — каждая со своей ловушкой.
- **Движок авторинга** `cabinet/engine/`: агент-АВТОР (промпт на принципах + контрастные ok/не-ok разной формы, БЕЗ скелета) → агент-КРИТИК (поштучный чеклист C1–C12, вердикт) → ≤1 правка → я. Grounding на эталонных `steps[]` (SymPy-проверены) — LLM делает только педагогику.
**Решение:** (1) НЕ Flutter Web — статичный Vite SPA легче в WebView. (2) Лесенка ≠ шаги решения: рунг вскрывает СКРЫТУЮ развилку (expert blind spot), а не переписывает видимую арифметику. (3) Анти-шаблон через ОТСУТСТВИЕ названного скелета в промпте + few-shot разной формы (называешь форму → модель её клонирует). (4) Движок — батч-офлайн (как декомпозиция), рантайм только читает.
**Итог:** ✅ кабинет билдится (tsc strict) + прогнан в Playwright (мобайл, 0 console-ошибок); движок прогнан на 3 задачах за **97 сек** — критик САМ поймал leading-вопрос на свежей задаче и автор переписал. Поймал и починил баг движка: жирное поле `reasoning` первым в structured-output → обрезка по токенам → «rungs missing» retry-loop (20–46 вызовов, 13 мин/задача). Фикс: reasoning не выводить, думать в reasoning-токенах.
