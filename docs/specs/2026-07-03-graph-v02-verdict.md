Полный проход по 118 узлам / 200 рёбрам / 118 привязкам к темам, с опорой на реальные задачи узлов.
Принципы: (1) ребро A→B живёт, только если БЕЗ навыка A типовые задачи B нерешаемы; (2) transitive reduction — прямое ребро убирается, если покрыто цепочкой; (3) привязка к теме — по фактическому содержанию задач, не по названию узла.
