## Примеры размётки

### Простая
**[FR03] ответ = 2/3**  (главный навык: `reduce_frac`)
  1. Найди НОД числителя и знаменателя: НОД(12, 18) = 6.  →  `6` `[gcd]` ✓
  2. Раздели числитель и знаменатель на 6: 12÷6 = 2, 18÷6 = 3. Получится 2/3.  →  `2/3` `[reduce_frac]` ✓

### Многошаговая (вот ради чего всё) — собрана из простых навыков + отпечатки
**[FR09] ответ = 3/4**  (главный навык: `order_of_ops`)
  1. Первая скобка: 5/6 − 1/3 = 5/6 − 2/6 = 1/2  →  `1/2` `[addsub_frac]` ✓
  2. Вторая скобка: 1/2 + 1/6 = 3/6 + 1/6 = 2/3  →  `2/3` `[addsub_frac]` ✓
  3. Делим первую скобку на вторую: 1/2 ÷ 2/3 = 1/2 × 3/2 = 3/4  →  `3/4` `[div_frac]` ✓
  Отпечатки ошибок:
    • ответ **1/3** → Умножил скобки вместо деления: 1/2 × 2/3 → `[div_frac]`

### Текстовая
**[FR11] ответ = 30000**  (главный навык: `frac_of_number`)
  1. На еду: 1/4 от 60000 = 15000  →  `15000` `[frac_of_number]` ✓
  2. Остаток после еды: 60000 − 15000 = 45000  →  `45000` `[int_add_sub]` ✓
  3. На транспорт: 1/3 от остатка 45000 = 15000  →  `15000` `[frac_of_number]` ✓
  4. Осталось: 45000 − 15000 = 30000  →  `30000` `[int_add_sub]` ✓
  Отпечатки ошибок:
    • ответ **25000** → Взял 1/3 на транспорт от всей зарплаты (60000), а не от остатка 45000 → `[frac_of_number]`
    • ответ **45000** → Остановился на остатке после еды, забыл вычесть транспорт → `[int_add_sub]`
